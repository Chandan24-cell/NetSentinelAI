## 🎯 Problem & Motivation
Traditional signature‑based intrusion detection systems fail to detect novel or zero‑day attacks. This project uses **machine learning** to recognise attack patterns from millions of real‑world network flows. It provides a complete, end‑to‑end solution that can be deployed in a Security Operations Center (SOC).

---

## ✨ Features
- **Multi‑class attack detection** – identifies 15 attack types (DDoS, DoS Hulk, PortScan, Bot, etc.)  
- **Live dashboard** – displays alerts, statistics, and attack timeline (auto‑refreshes)  
- **Top attackers list** – shows the most active malicious IPs with block buttons  
- **IP blocking** – one‑click simulated (or actual) blocking of attacking IPs  
- **Upload & detect** – process any CSV file generated by CICFlowMeter  
- **High performance** – trained on 2.8 million flows, near‑perfect detection for common attacks  

---

## 📊 Dataset

We used the **CICIDS2017 dataset**, a benchmark for intrusion detection research.

- **Total flows:** 2,830,743  
- **Features:** 78 network flow characteristics  
- **Classes:** 15 (BENIGN + 14 attack types)

### Attack Distribution

| Attack Type | Number of Flows |
|-------------|-----------------|
| BENIGN | 2,271,320 |
| DoS Hulk | 230,124 |
| PortScan | 158,804 |
| DDoS | 128,025 |
| ... | ... |

> The dataset is **highly imbalanced**, but the model still performs excellently on majority classes.

## ⚙️ Installation

### Prerequisites
- Python 3.11 or higher
- Git (optional)

### Steps
1. **Clone the repository**  
   ```bash
   git clone https://github.com/Chandan24-cell/NetSentinelAI.git
   cd itrix-ai-ids
Create and activate a virtual environment

bash
python -m venv venv
source venv/bin/activate      # Linux/macOS
venv\Scripts\activate         # Windows
Install dependencies

bash
pip install pandas numpy scikit-learn flask joblib
Download the trained model (if not included) – the repository already contains ml/random_forest_full.pkl.

Prepare the dataset (optional) – the CICIDS2017 CSV files are not included due to size; you can download them from the official source.

🚀 Usage
Start the Dashboard
bash
cd backend
python app.py
Open your browser at http://localhost:5001.

Detect Attacks
Click Choose file and select a CSV file (from the dataset or generated by CICFlowMeter).

Click Upload CSV → the file is saved to the uploads/ folder.

Click Run Detection → the prediction script processes the file and writes alerts to alerts.json.

Wait a few seconds – the dashboard auto‑refreshes and displays the alerts.

Block an IP
In the Top Attackers list or the Alerts Table, click the Block button next to an IP.

The backend will simulate blocking (or actually block if you configure pf).

A confirmation message appears.

## 📈 Model Performance

| Metric                  | Value |
|-------------------------|-------|
| Overall Accuracy        | ~96%* |
| Weighted Avg F1-Score   | 1.00 |
| Macro Avg F1-Score      | 0.82 |

\* The overall accuracy is inflated due to the majority class (**BENIGN**).  
The model performs excellently on common attacks such as **DDoS, DoS Hulk, and PortScan**, and adequately on rarer ones.
Dataset: CICIDS2017  
Total flows used for evaluation: ~2.8M
Model: Random Forest
